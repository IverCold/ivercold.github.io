Numenera 2.5 Crafting System


Crafting Process

Crafting is a series of Intellect based tasks. Generally whole process includes two steps:

  1. Project assessed difficulty calculation. Any object or structure you want create has assessed difficulty level that determines amount of subtasks you must complete to finish crafting.
    Final assessed difficulty formula: minimal crafting level + project type modifier + "hindered" modifiers - level of appropriate crafting skill - assets - "eased" modifiers.
    Project type modifiers presented in table below.
  2. Performing craft subtasks. Final assessed difficulty level is equal to amount of subtasks that must be performed to complete crafting. First subtask will have first difficulty level. Difficulty level of each next subtask will increase by 1. Subtask difficulty level can be decreased by appropriate Crafting skill, assets and "eased" modifiers. Efforts can be used only if subtask duration is no longer than 1 day.
    If at any point the crafter fails on a single subtask, the object or structure isnâ€™t ruined. Instead, the crafter merely loses time. A failure means the time spent on that crafting roll was wasted, but the character can spend that much time again and then try to succeed at that same subtask. If the crafter fails twice in a row on the same subtask, the character can continue crafting but in addition to losing another interval of crafting time, 1 unit of the highest level material being used is destroyed in a mishap and must be replaced before crafting can continue.
    Subtask duration formula: project crafting time * subtask difficulty level / all subtasks difficulty levels sum.
    Project crafting time presented in table below. Total project crafting time can't be less than half of time for minimal crafting level and less than time for first crafting level.

The appropriate crafting skill also applies to any task to repair objects or structures. Similarly, mundane crafting skills apply to the repair of associated items. In either case, if a Wright attempts to repair an object or structure they personally crafted, they gain an asset to the repair task.
